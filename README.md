## Project Overview
This repository is a comprehensive **Digital Literacy Portfolio** created as part of the CSE0001 course. As a designated Student Digital Ambassador, I have developed this resource to demonstrate core competencies in digital communication, professional branding, and online safety. The project serves as a guide for peers to navigate the digital landscape effectively and securely.

---

## Module Summaries

| Task | Focus Area | Key Deliverables |
| :--- | :--- | :--- |
| Task 1 | Digital Literacy Awareness | A visual infographic designed in Canva highlighting safe internet practices and the definition of digital literacy. |
| Task 2 | Professional Digital Portfolio | Setup of professional profiles on GitHub, LinkedIn, and Kaggle to establish a verified academic and career presence. |
| Task 3 | Coding & Collaboration | Completion of a HackerRank coding challenge and the creation of a 5-question Digital Literacy Quiz via Google Forms. |
| Task 4 | Professional Communication | Drafts of formal academic and professional emails, alongside a "Do’s and Don'ts" guide for social media etiquette. |
| Task 5 | Cybercrime Awareness | A detailed case study on UPI fraud and an 8-step prevention checklist for staying safe online in India. |
